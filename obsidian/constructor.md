构造器的常见应用场景

创建对象时，同时对对象成员变量初始化赋值。

如果不写构造器，Java 是会为类自动生成一个无参构造器的。

一旦定义了有参数构造器， Java 就不会帮我们的类自动生成无参构造器了，此时手写一个无参数构造器
